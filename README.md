# CAPTCHA-f

## Overview
CAPTCHA-f is a cybersecurity framework that evaluates the effectiveness of CAPTCHA mechanisms against automated bot attacks using OCR-based techniques.

## Objectives
- Assess CAPTCHA resilience against automation
- Measure OCR success rates
- Analyze security weaknesses
- Recommend stronger CAPTCHA designs

## Technologies Used
- Python
- Flask
- Selenium
- Tesseract OCR
- Pillow (PIL)
- CSV

## Methodology
1. CAPTCHA generation
2. Automated collection
3. Image preprocessing
4. OCR solving
5. Accuracy evaluation
6. Security analysis

## Metrics
- Success Rate
- Character Accuracy
- Solve Time

## Findings
Text-based CAPTCHAs with minimal distortion showed significantly higher OCR success rates compared to more complex variants.
